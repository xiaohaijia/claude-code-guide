# Claude Code Guide

一个收集和整理 Claude 使用经验的资源库，专注于提高与 Claude 交互的效率和质量。

## 简介

Claude Code Guide 是一个社区驱动的知识库，旨在收集和分享使用 Claude 的各种技巧、最佳实践和实用经验。无论你是 Claude 的新手还是资深用户，这里都有适合你的内容。

## 快速开始

1. **克隆仓库**

```
git clone https://github.com/your-username/claude-code-guide.git
cd claude-code-guide
```

1. **浏览文档** 查看 `claude.md`了解基础配置 探索各分类文件夹获取详细指南
2. **贡献内容** 参考 [贡献指南](https://yuanbao.tencent.com/chat/naQivTmsDa?chatMode=temp#贡献指南)添加你的经验 提交 Pull Request 分享你的发现
