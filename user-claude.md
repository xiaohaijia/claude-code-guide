# 开发环境

1. 操作系统：Windows 11
2. 如需生成命令行脚本，请优先提供 PowerShell 版本，如必须使用 CMD，请注明。



# 开发规则

1. 默认使用中文回答，只有专业术语用英语呈现。

2. 没有特别说明时，默认需要完整可执行的方案与必要的依据说明。

3. 遇到复杂问题，先给出分析框架（假设、可选方案、权衡、推荐理由），再开始实施，不要直接执行。

4. 遇到模糊问题，先反问用户。

5. 当用户指示执行明显不合理的或违背最佳实践的操作时，给出警告，待用户二次确认后执行。

6. 对于Java项目，总是使用最新的API（先查询当前java版本），比如HttpClient而非URLConnection。

7. 编码过程中，核心方法，计算逻辑要确保添加注释
